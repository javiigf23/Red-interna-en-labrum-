# ESTADO DEL ARTE

Una red de área local (LAN, por sus siglas en inglés Local Area Network) es una infraestructura de comunicaciones que permite la interconexión de dispositivos dentro de un área limitada, como una oficina o edificio, para compartir datos y recursos de forma rápida y eficiente. En entornos empresariales, las LAN sirven como la base de la conectividad interna entre ordenadores, servidores, impresoras y otros equipos de red, facilitando la colaboración y el acceso a servicios centralizados.

Las LAN pueden ser cableadas (usando tecnologías como Ethernet) o inalámbricas (WLAN, basadas en Wi-Fi), y suelen incluir conmutadores, routers y puntos de acceso que gestionan el tráfico de datos. Esto permite compartir recursos como archivos, aplicaciones y hardware de red, así como proporcionar acceso a Internet de manera conjunta, lo que resulta esencial para la productividad en las organizaciones.

Las redes LAN empresariales modernas se diseñan para ofrecer altas velocidades de transmisión y flexibilidad, combinando soluciones cableadas e inalámbricas según las necesidades de movilidad y densidad de usuarios. Además, estas redes incorporan mecanismos de seguridad que protegen la comunicación interna mediante configuraciones de dispositivos y control de acceso.

En resumen, las redes internas de empresa representan una parte fundamental de la infraestructura TI al garantizar la conectividad eficiente, el intercambio de recursos y la seguridad dentro de un entorno corporativo, lo cual es indispensable para el funcionamiento diario de los procesos organizativos.
